# Jacob Arthurs - Portfolio

A modern, responsive portfolio website showcasing my professional experience, technical projects, and expertise as a Senior Full Stack Software Engineer.

## Overview

This portfolio serves as a comprehensive digital presence where I present my background in full-stack development. The site demonstrates both my technical capabilities and commitment to clean, user-focused design.

## Technology Stack

- **Framework:** Angular
- **UI Components:** Angular Material
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Build Tool:** Angular CLI
- **Deployment:** Cloudflar

## Key Features

- Fully responsive design optimized for all device sizes
- Professional experience timeline with detailed role descriptions
- Technical skills and certifications showcase
- Project portfolio with live demonstrations
- Integrated contact form for professional inquiries
- Optimized dark theme for enhanced readability

## Architecture

Built with modern web development best practices, this application leverages Angular's component-based architecture, reactive forms, and routing capabilities. The design system combines Angular Material components with custom Tailwind CSS utilities to achieve a polished, professional aesthetic.

## Purpose

This portfolio represents my professional identity and technical proficiency in software engineering. It serves multiple objectives:

- Demonstrating practical Angular and TypeScript expertise
- Showcasing design sensibility and attention to user experience
- Providing a centralized platform for professional networking
- Highlighting career achievements and technical competencies

## Local Development

```bash
# Install dependencies
npm install

# Run development server
ng serve

# Build for production
ng build --configuration production
```
